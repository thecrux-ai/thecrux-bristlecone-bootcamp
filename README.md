# Bristlecone × theCRUX.ai - Claude Code Bootcamp

Welcome. This repo is your workspace and guide for the two day Claude Code bootcamp.

## Start here

Open **[student-handbook.md](student-handbook.md)**. It is your map for both days, with a link to every session.

## What's inside

- `student-handbook.md` - the agenda and session index
- `session-0` to `session-5` - one file per session, with every prompt you need
- `capstone.md` - your own build on Day 2
- `closing.md` - wrap up and what to do after
- `CLAUDE.template.md` - a starter for your own context file
- `appendix.md` - general troubleshooting
- `sample-data/` - the data we use in the Day 2 build
- `.claude/skills/` - Anthropic's document skills (Word, PowerPoint, PDF, Excel). With these in place, Claude builds polished documents and decks for you out of the box, no setup needed.

## Getting started

Starting Claude is four clicks in Visual Studio Code:

1. Open **Visual Studio Code**.
2. **File → Open Folder...** and pick `thecrux-bristlecone-bootcamp`.
3. **Terminal → New Terminal** (a panel opens at the bottom).
4. Type `claude` and press Enter.

Then open **[student-handbook.md](student-handbook.md)** and start with Session 0, which walks you through this step by step.
